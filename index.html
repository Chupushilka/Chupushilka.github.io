Сырки, которые изменили мою жизнь

В современном мире развивается всё - технологии, образ мыслей, люди. Неизменным остается только одно - сырки. Для каждого из нас глазированный сырок - это что-то особенное, сокральное, личное, для некоторых даже интимное.
